@ vue, vite, pinia:
------------------------------------------------------------------
@ start project:

- first steps:

$ yarn create vite // create project
$ yarn // install dependencies
$ yarn dev // start project

- install vue-router and pinia:

$ yarn add vue-router@4 pinia

- eslint & prettier:

$ yarn add -D eslint eslint-config-prettier eslint-plugin-vue prettier vite-plugin-eslint vue-eslint-parser
